YAML格式的Service定义文件的完整内容如下：

```yaml
apiVersion: v1          # Required
kind: Service           # Required
metadata:               # Required
  name: string          # Required
  namespace: string     # Required
  labels: 
    - name: string
  annotations:
    - name: string
spec:                   # Required
  selector: []          # Required
  type: string
  - port: 8080
    protocol: TCP
    targetPort: 8080
  selector:
    tier: frontend
  sessionAffinity: None
  type: ClusterIP
status:
  loadBalancer: ()
```
