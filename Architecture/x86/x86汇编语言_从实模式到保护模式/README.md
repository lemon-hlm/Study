《x86汇编语言 从实模式到保护模式》读书笔记

本书采用开源的 NASM 汇编语言编译器和VirtualBox虚拟机软件，以Intel处理器为基础，详细讲解了Intel处理器的指令系统和工作模式，以大量的代码演示了16／32／64位软件的开发方法。集中介绍处理器的16位实模式和32位保护模式，以及基本的指令系统。
