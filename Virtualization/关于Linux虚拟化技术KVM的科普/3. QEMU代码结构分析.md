# 1 代码结构

如我们所知，QEMU是一个模拟器，它能够动态模拟**特定架构的CPU指令**，如X86，PPC，ARM等等。

QEMU模拟的架构叫**目标架构**，运行QEMU的系统架构叫主机架构，QEMU中有一个模块叫做微型代码生成器（TCG），它用来将目标代码翻译成主机代码。如下图所示。
