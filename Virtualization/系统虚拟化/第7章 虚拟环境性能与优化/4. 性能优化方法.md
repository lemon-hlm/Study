# 0 概述

无论什么虚拟化技术, 相对于物理机而言, 处理器在Hypervisor上执行开销都将构成额外的虚拟化开销. 处理器在Hypervisor上执行的时间越长, 虚拟化开销越高, 虚拟环境的吞吐量和性能就越差, 反之则虚拟环境的性能越好. 

因此, 优化的重点就应该考虑如何降低这种额外开销. 

虚拟环境性能优化方法很多, 而且是具体问题具体分析. 下面归纳一些以往优化中常用方法, 供参考.
